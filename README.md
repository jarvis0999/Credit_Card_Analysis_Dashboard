# Credit_Card_Analysis_Dashboard
Credit Card Analysis and Dashboard creation using Power BI and SQL 

To design and implement a dynamic weekly dashboard for credit card operations that delivers real-time insights into critical performance metrics and trends, empowering stakeholders to effectively monitor and analyze credit card performance.
We Analyse and create two dashboards one Credit Card Financial Dashboard Transaction which shows the overall transaction report of the provider and another Credit Card Financial Dashboard Customer which gives the customer's usage insights and visualisations of the card
